AVSL_SLF4J_Logger 

class AVSL_SLF4J_Logger extends MarkerIgnoringBase

A trait that, mixed in, provides SLF4J methods for an AVSL Logger class. Kept separate, for clarity. Marker methods ignore the marker parameters.
